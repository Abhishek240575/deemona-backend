📋 Setup Guide
1
Add environment variables to Render
Go to Render → deemona-backend → Environment and add these variables:
QBO_CLIENT_ID=your_quickbooks_client_id QBO_CLIENT_SECRET=your_quickbooks_client_secret QBO_REDIRECT_URI=https://deemona-backend.onrender.com/v1/integrations/quickbooks/callback QBO_ENVIRONMENT=production QBO_WEBHOOK_VERIFIER_TOKEN=your_webhook_token XERO_CLIENT_ID=your_xero_client_id XERO_CLIENT_SECRET=your_xero_client_secret XERO_REDIRECT_URI=https://deemona-backend.onrender.com/v1/integrations/xero/callback XERO_WEBHOOK_KEY=your_xero_webhook_key
2
Create QuickBooks app (takes 5 minutes)
Go to developer.intuit.com → Create an App → QuickBooks Online → Add redirect URI from above → Copy Client ID and Secret
3
Add route to your backend index.js
// src/routes/index.js — add these 3 lines: const erp = require('./integrations/finance-erp'); const erpSync = require('../jobs/erp-sync'); router.use('/integrations', erp); erpSync.start(); // starts hourly + daily sync jobs
4
Run schema patch to add external_id columns
-- Run this SQL in psql against deemona_enterprise: ALTER TABLE invoices ADD COLUMN IF NOT EXISTS external_id VARCHAR(255); ALTER TABLE invoices ADD COLUMN IF NOT EXISTS external_source VARCHAR(50); ALTER TABLE invoices ADD COLUMN IF NOT EXISTS customer_name VARCHAR(255); ALTER TABLE gl_accounts ADD COLUMN IF NOT EXISTS external_id VARCHAR(255); ALTER TABLE gl_accounts ADD COLUMN IF NOT EXISTS external_source VARCHAR(50); ALTER TABLE gl_accounts ADD COLUMN IF NOT EXISTS balance DECIMAL(18,2) DEFAULT 0; ALTER TABLE suppliers ADD COLUMN IF NOT EXISTS external_id VARCHAR(255); ALTER TABLE suppliers ADD COLUMN IF NOT EXISTS external_source VARCHAR(50); CREATE UNIQUE INDEX IF NOT EXISTS idx_inv_ext ON invoices(tenant_id, external_id, external_source); CREATE UNIQUE INDEX IF NOT EXISTS idx_acct_ext ON gl_accounts(tenant_id, external_id, external_source); CREATE UNIQUE INDEX IF NOT EXISTS idx_sup_ext ON suppliers(tenant_id, external_id, external_source); CREATE UNIQUE INDEX IF NOT EXISTS idx_po_ext ON purchase_orders(tenant_id, external_id, external_source); ALTER TABLE kpi_values ADD CONSTRAINT kpiv_unique UNIQUE (tenant_id, kpi_definition_id, recorded_at);
5
Install required npm package
npm install node-cron axios
6
Connect your ERP above and trigger first sync
Click "Connect QuickBooks" or "Connect Xero" above. After OAuth completes, the first full sync runs automatically. Live KPIs will appear in all Finance dashboards within 2–3 minutes.
