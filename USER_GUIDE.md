## 📊 Dashboard Features

### Viewing a Dashboard

1. Click any dashboard card
2. Modal opens with 3 tabs:
   - **📊 Metrics**: Key performance indicators
   - **📈 Charts**: Visual data representations
   - **ℹ️ Info**: Dashboard details

### Metrics Tab

- Displays 3-6 key metrics
- Shows current value
- Indicates change vs. prior period
- Color-coded: Green (↑) / Red (↓)

### Charts Tab

- Line charts for trends
- Bar charts for comparisons
- Doughnut charts for distributions
- Interactive hover tooltips

### Source Data Viewer

1. Click "🗄️ View Source Data"
2. Full table view with pagination
3. Search and filter capabilities
4. Sort by any column
5. Export to CSV

### Exporting Data

1. Click "⬇️ Export" button
2. Choose format: CSV or JSON
3. File downloads automatically

## 📤 Data Upload

### Upload Your Own Data

1. Click "📤 Upload Data" in top controls
2. Drag & drop or browse for file
3. Supported formats: CSV, XLSX, XLS
4. Preview data before confirming
5. Click "✓ Use This Data"

### Data Requirements

**Format Guidelines:**
- First row should contain headers
- Numeric values for metrics
- Date columns for trends
- No empty rows

**Best Practices:**
- Clean data before upload
- Use consistent date formats
- Remove special characters
- Limit file size to 10 MB

## 💡 Tips & Best Practices

### Performance Tips

1. **Regular Data Refresh**: Click refresh for latest data
2. **Close Unused Modals**: Better performance
3. **Limit Open Charts**: Close charts not in use
4. **Clear Browser Cache**: If experiencing issues

### Data Best Practices

1. **Upload Clean Data**: Remove duplicates and errors
2. **Use Consistent Formats**: Dates, numbers, text
3. **Regular Backups**: Export data periodically
4. **Verify After Upload**: Check preview before confirming

### Security Best Practices

1. **Strong Passwords**: Use complex, unique passwords
2. **Logout When Done**: Especially on shared devices
3. **Regular Updates**: Keep browser updated
4. **Review Access**: Check who has access regularly

### Dashboard Organization

1. **Use Favorites**: Pin frequently used dashboards
2. **Role Filters**: Filter by your role for relevant views
3. **Search Function**: Quick access to specific dashboards
4. **Category Grouping**: Organize by business function

## 🔧 Troubleshooting

### Common Issues

**Dashboard won't load:**
- Check internet connection
- Refresh page (F5)
- Clear browser cache
- Try different browser

**Data upload fails:**
- Check file format (CSV/XLSX)
- Verify file size < 10 MB
- Check column headers
- Remove special characters

**Charts not displaying:**
- Wait for data to load
- Check if data is available
- Try switching tabs
- Refresh dashboard

**Mobile menu won't open:**
- Tap hamburger icon ☰
- Swipe from left edge
- Check screen orientation
- Restart browser app
